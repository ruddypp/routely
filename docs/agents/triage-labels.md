# Agent Triage Labels

Use these canonical labels when triaging or creating issues for Routely.

| Role | Label |
| --- | --- |
| Needs maintainer evaluation | `needs-triage` |
| Waiting on reporter/user input | `needs-info` |
| Ready for an agent to implement | `ready-for-agent` |
| Ready for human implementation | `ready-for-human` |
| Will not be actioned | `wontfix` |

If the label does not exist in GitHub, mention that it may need to be created. Do not create duplicate near-synonyms.
